# GPT_Web
- 一个可与GPT聊天的网页客户端
- 前往https://platform.openai.com/account/org-settings 获取你的组织ID
- 前往https://platform.openai.com/account/api-keys 获取你的APIkey

# 使用(nodejs)
- 下载以下模块
```
npm install express
npm install openai
npm install chatgpt
```
- 将源代码下载至本地
```
git clone https://github.com/DINOREXNB/GPT_Web.git
```
- 进入工作区后，执行
```
node server.js
```
